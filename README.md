
# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview
This project analyzes pizza sales data using SQL to uncover valuable business insights. The analysis focuses on customer ordering behavior, revenue generation, product performance, and sales trends.

## 🎯 Objectives
- Analyze pizza sales performance.
- Identify best-selling pizza types and sizes.
- Calculate revenue and order trends.
- Answer business-related questions using SQL queries.
- Generate actionable insights for decision-making.

## 🛠️ Tools Used
- MySQL
- SQL Queries
- Canva (for presentation)

## 📂 Dataset Information
The project uses four tables:

1. **orders** – Order details including date and time.
2. **order_details** – Quantity of pizzas ordered.
3. **pizzas** – Pizza size and price information.
4. **pizza_types** – Pizza category and type information.

## 📊 Business Questions Solved

### Basic Analysis
1. Retrieve the total number of orders placed.
2. Calculate the total revenue generated from pizza sales.
3. Identify the highest-priced pizza.
4. Identify the most commonly ordered pizza size.
5. List the top 5 most ordered pizza types along with quantity sold.

### Intermediate Analysis
6. Find the total quantity sold for each pizza category.
7. Determine the distribution of orders by hour of the day.
8. Analyze category-wise pizza distribution.

### Advanced Analysis
9. Calculate the average number of pizzas ordered per day.
10. Determine the top 3 pizza types based on revenue generated.

## 🔍 Key Insights
- Large-sized pizzas were the most preferred by customers.
- A small number of pizza types contributed significantly to total revenue.
- Customer orders were concentrated during peak meal hours.
- Certain pizza categories consistently outperformed others in quantity sold.
- Top-selling pizzas generated a major share of overall revenue.

## 📈 Skills Demonstrated
- SQL Joins
- Aggregate Functions
- Group By & Order By
- Revenue Analysis
- Business Intelligence
- Data Exploration
- Trend Analysis

## 🚀 Project Outcome
This project demonstrates how SQL can transform raw sales data into meaningful business insights. The findings can help restaurants improve inventory planning, menu optimization, and sales strategies.
